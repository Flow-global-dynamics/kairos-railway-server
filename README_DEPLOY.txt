# FlowGlobal V1 Kairos - Déploiement Railway

Étapes :
1. Aller sur https://railway.app
2. Cliquer sur "New Project" > "Deploy from Template"
3. Choisir "Upload ZIP" et envoyer ce fichier
4. Aller dans Settings > Variables d'environnement :
   - ACCESS_TOKEN = FLOWGLOBAL2025
   - BINANCE_API_KEY = VOTRE_CLÉ
   - BINANCE_API_SECRET = VOTRE_SECRET
   - MODE_SANDBOX = true

API accessible à /status, /flowbytebeat/trigger, /flowsniper/trigger, etc.
Cockpit FlowGlobal prêt à connecter via app iOS.
